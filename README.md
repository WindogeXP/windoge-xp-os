# Windoge XP (OS)

A browser-based operating system bringing early internet nostalgia to the Web3 era. Run classic apps, games, and on-chain experiences directly in your browser.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Code Quality**: ESLint + Prettier
